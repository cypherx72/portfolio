# About Tavonga Chitambira

## 👋 Hi, I'm Tavonga

I'm a full-stack developer passionate about building beautiful, interactive, and performant web applications. I specialize in modern frontend frameworks and create seamless user experiences with clean, maintainable code.

## 🎯 What I Do

I focus on crafting:
- **Dynamic web applications** using React and Next.js
- **Pixel-perfect UI/UX** with Tailwind CSS and modern design systems
- **Smooth animations and interactions** with Framer Motion
- **End-to-end solutions** from concept to deployment on Vercel

My approach combines technical excellence with a strong emphasis on user experience and performance.

## 🛠️ Tech Stack

### Frontend
- **Framework:** Next.js 16, React 19
- **Styling:** Tailwind CSS 4, shadcn/ui
- **Animation:** Framer Motion 12
- **Icons:** react-icons, lucide-react, @hugeicons/react
- **Language:** TypeScript

### Extras
- Lottie animations
- Dark mode support (next-themes)
- Analytics and monitoring

### Deployment
- **Platform:** Vercel
- **Version Control:** Git & GitHub
