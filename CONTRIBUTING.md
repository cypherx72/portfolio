# Contributing

This is a personal portfolio, so contributions are limited in scope — but issues and suggestions are welcome.

## What's welcome

- Bug reports (broken layout, animation glitches, broken links)
- Typo or copy fixes
- Accessibility improvements
- Performance suggestions

## What's not

- Feature requests that change the personal content or identity of the site
- Redesigns or major UI overhauls

## How to open an issue

1. Check if the issue already exists
2. Open a new issue with a clear title and description
3. Include browser/OS info for visual bugs

## Local setup

```bash
git clone https://github.com/cypherx72/portfolio.git
cd portfolio
npm install
npm run dev
```

No environment variables are required for local development.
